# A股股票量化分析和模拟炒股系统

这是一个基于Python的A股股票量化分析系统，能够根据行业自动选择股票，进行技术分析、预测和模拟交易。

## 功能特性

### 🔍 量化分析
- **技术指标计算**：SMA20、RSI14、MACD
- **价格预测**：基于线性回归预测未来5天价格
- **多维度分析**：
  - 大盘走势（上证指数）
  - 行业趋势
  - 公司财报（净利润）

### 📊 智能建议
- 综合评分系统（满分7分）
- 买卖建议：强烈买入/买入/观望/卖出
- 持有时间建议

### 🎯 模拟交易
- 自动根据建议执行买卖
- 资金管理（初始10万元）
- 持仓跟踪和盈亏计算

### 🏭 行业选择
- 支持机械设备、半导体、新能源等行业
- 自动从行业成分股中选择2支股票
- 随机选择确保多样性

## 安装依赖

```bash
pip install akshare scikit-learn ta pandas numpy
```

## 使用方法

### 运行分析
```bash
python stock.py
```

程序会：
1. 自动选择行业（当前硬编码为"机械设备"）
2. 获取行业股票列表
3. 随机选择2支股票
4. 对每支股票进行完整分析
5. 根据建议执行模拟交易
6. 显示最终资产状况

### 修改行业
在 `stock.py` 中修改 `industry_input` 变量：
```python
industry_input = "半导体"  # 或 "新能源"
```

支持的行业：
- 机械设备
- 半导体
- 新能源
